--------------------------------------------------
📌 PROJECT OVERVIEW
--------------------------------------------------
This is a modern, responsive, and animated personal portfolio website
developed using HTML, CSS, and JavaScript.

The portfolio showcases my skills, projects, certifications, internship
experience, and achievements in a professional and visually appealing way.
It is designed to be recruiter-friendly and suitable for placements,
internships, and professional networking.

--------------------------------------------------
✨ FEATURES
--------------------------------------------------
• Modern dark UI with liquid crystal / glassmorphism design  
• Custom "SK" favicon visible in browser tab  
• Smooth scroll animations and section transitions  
• Interactive micro-interactions and hover effects  
• Animated buttons with ripple and glow effects  
• Programming language logos using icons  
• Project cards with GitHub redirect buttons  
• Responsive design (mobile, tablet, desktop)  
• Clean and well-structured code  

--------------------------------------------------
🛠 TECHNOLOGIES USED
--------------------------------------------------
• HTML5  
• CSS3 (Animations, Transitions, Glassmorphism)  
• JavaScript (DOM manipulation, scroll effects)  
• Font Awesome / Devicon (Icons)
